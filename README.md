# 💻 Gerenciador de Empresas

Projeto desenvolvido durante o curso de **Servlets e JSP (Alura)**, com o objetivo de praticar conceitos fundamentais de desenvolvimento web em Java.

---

## 🚀 Funcionalidades

- Cadastro de empresas
- Listagem de empresas
- Remoção de empresas
- Alteração de dados
- Navegação entre páginas com Servlets e JSP

---

## 🛠️ Tecnologias utilizadas

- Java
- Servlets
- JSP
- JSTL
- Apache Tomcat
- Eclipse IDE

---

## 📂 Estrutura do projeto

O projeto segue uma arquitetura simples baseada em:

- **Servlets** → controle das requisições
- **JSP** → camada de visualização
- **Classe Banco (simulada)** → armazenamento em memória
